# JHELIFIRE
## ESECUZIONE APPLICAZIONE

All'interno della cartella dist troverete il jar dell'applicazione (helifire.jar).

### Metodo 1: Doppio clic
- Fare doppio clic sul file `helifire.jar` per avviare l'applicazione.

### Metodo 2: Terminale
- Aprire un terminale.
- Posizionarsi nella cartella dist usando il comando:
  ```bash
  cd /percorso/alla/cartella/dist
  ```
- Eseguire il comando:
  ```bash
  java -jar helifire.jar
  ```

## COMANDI PER MUOVERSI:

Utilizzare le frecce direzionali per muoversi a sinistra e a destra.

## COMANDI PER SPARARE:

Utilizzare la barra spaziatrice per sparare.

## Caratteristiche del Gioco

- Sistema di Livelli Progressivi: La difficoltà aumenta con il progredire dei livelli.
- Nemici Diversificati:
  - GreenHeli: Elicotteri base con pattern di sparo semplice.
  - BlueHeli: Dal livello 4 utilizzano un pattern di sparo a ventaglio.
  - YellowHeli: Dal livello 4 sparano raffiche di proiettili.
  - ShipDestroyer: Navi speciali con diverse velocità in base al livello.
- Sistema di Bonus:
  - Se il giocatore ha tutte le vite: +500 punti.
  - Se il giocatore ha perso vite: recupero di una vita.
- Sistema di Punteggi e High Scores.
- Audio ed Effetti Sonori: È possibile attivare o disattivare l'audio dalla schermata delle opzioni.
